# Fit-News
Get the most recent news in the muscle and fitness world!

## Deployed on Heroku!
[Fit-News](https://fit-news.herokuapp.com/)

## Technologies Used:
Node.js, Express.js, MongoDB, Mongoose, HTML5, CSS3, and Bootstrap.
